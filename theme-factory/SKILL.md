# 主题工厂技能

此技能提供精选的专业字体和颜色主题集合，每个主题都包含精心选择的调色板和字体配对。一旦选择了主题，就可以将其应用于任何工件。

## 目的

要将一致、专业的样式应用于演示幻灯片，请使用此技能。每个主题包括：
- 具有十六进制代码的连贯调色板
- 标题和正文的互补字体配对
- 适合不同上下文和受众的独特视觉标识

## 使用说明

要将样式应用于幻灯片或其他工件：

1. **显示主题展示**：显示 `theme-showcase.pdf` 文件，让用户可视化地查看所有可用主题。不要对其进行任何修改；只需显示文件以供查看。
2. **询问他们的选择**：询问将哪个主题应用于幻灯片
3. **等待选择**：获取关于所选主题的明确确认
4. **应用主题**：一旦选择了主题，将所选主题的颜色和字体应用于幻灯片/工件

## 可用主题

以下 10 个主题可用，每个都在 `theme-showcase.pdf` 中展示：

1. **Ocean Depths** - 专业和舒缓的海洋主题
2. **Sunset Boulevard** - 温暖和充满活力的日落颜色
3. **Forest Canopy** - 自然和接地气的大地色调
4. **Modern Minimalist** - 干净和当代的灰度
5. **Golden Hour** - 丰富和温暖的秋季调色板
6. **Arctic Frost** - 凉爽和清脆的冬季主题
7. **Desert Rose** - 柔和和精致的尘土色调
8. **Tech Innovation** - 大胆和现代的技术美学
9. **Botanical Garden** - 新鲜和有机的花园颜色
10. **Midnight Galaxy** - 戏剧性和宇宙的深色调

## 主题详细信息

每个主题在 `themes/` 目录中定义，包含完整规范，包括：
- 具有十六进制代码的连贯调色板
- 标题和正文的互补字体配对
- 适合不同上下文和受众的独特视觉标识

## 应用过程

选择首选主题后：
1. 从 `themes/` 目录读取相应的主题文件
2. 在整个幻灯片中一致地应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片上保持主题的视觉标识

## 创建您自己的主题
要处理现有主题都不适用于工件的情况，创建自定义主题。基于提供的输入，生成一个类似于上面的新主题。给主题一个类似的名称，描述字体/颜色组合代表什么。使用提供的任何基本描述来选择适当的颜色/字体。生成主题后，显示以供审查和验证。之后，如上所述应用主题。
